Instructions

To begin the game, type the start command

To exit the game, use the finish command

You can move only through unlocked rooms (transparent)

Locked rooms are not accessible (solid)

Navigation

up : Move one floor up

down : Move one floor down

north : Move towards front

south : Move towards back

east : Move towards right

west : Move towards left

Interaction

say <your message> : To send message to everyone in the room

yell <your message> : To send message to everyone in the world

tell <player name> <your message> : To send message to specific user in the room

World Intro

Room : {a user_location a}
